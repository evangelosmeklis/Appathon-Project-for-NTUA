This web application provides you with the opportunity to search through data gathered by clinicaltrials.gov . You will be able to search for the 10 most used medicines for any disease that the above website has gathered information about. Moreover, you will have the opportunity to view the value range of the dosage for each medicine of the above. Furthermore, you can refine your search by using a filter based on the starting date of the clinical trial. This project was created for the Internet and Applications Course of the ECE NTUA.
