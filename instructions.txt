Disclaimer: Το project στήθηκε σε περιβάλλον Ubuntu 18.08 (Υπάρχει η δυνατότητα τρεξίματος του σε Windows, ωστόσο οι οδηγίες είναι για το πως στήθηκε σε Ubuntu)

1. Για το κατέβασμα των δεδομένων.

Το κατέβασμα των δεδομένων γίνεται από το εξής link: https://clinicaltrials.gov/AllPublicXML.zip

Kάνουμε unzip τον φάκελο αυτό

To αρχείο αυτό περιέχει αρκετούς φακέλους με χιλιάδες αρχεία AllPublicXML

2. Για το parsing των αρχείων XML 

To parsing γίνεται με το πλήρες πακέτο της εφαρμογής BaseX το οποίο διατίθεται δωρεάν στο συγκεκριμένο link : http://files.basex.org/releases/9.4/BaseX94.zip

Κάνοντας unzip, ανοίγουμε το αρχειο BaseX.jar το οποίο ανοίγει και το γραφικό περιβάλλον της εφαρμογής

Στο γραφικό περιβάλλον πατάμε στο μενού πάνω, Database-> New

Κατευθυνόμαστε στον φάκελο AllPublicXML και τον επιλέγουμε

Αν πατήσουμε ΟΚ δημιουργείται η βάση δεδομένων που θέλουμε

3. Τρέχουμε το API που συνδέεται με την βάση που μόλις δημιουργήσαμε

Κατευθυνόμαστε στον φάκελο του basex που κατεβάσαμε από το link

Ανοίγουμε τον φάκελο bin

Ανοίγουμε ένα command window μέσα στον φάκελο αυτό και τρέχουμε την εντολή: ./basexhttp

Αφήνουμε το command window ανοιχτό και τώρα το API είναι λειτουργικό

4. Για την εκτέλεση των Queries

Για να εκτελεστούν τα Queries αντιγράφουμε το αρχείο mytest.xq το οποίο βρίσκεται στον φάκελο με όνομα basex_files του github repository της εργασίας

Το αρχείο αυτό το κάνουμε επικόλληση στον φάκελο basex/webapp του φακέλου που κατεβάσαμε από το link

5. Άνοιγμα της εφαρμογής

Ανοίγουμε το αρχείο intro.html σε έναν browser (στην ανάπτυξη χρησιμοποιήθηκε το firefox, αλλά μπορεί και σε άλλον)

6. Δοκιμή της εφαρμογής

Η δοκιμή της εφαρμογής γίνεται τώρα αποκλειστικά στον browser, δεν χρειάζεται κάποιο άλλο βήμα.
